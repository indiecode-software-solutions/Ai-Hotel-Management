## ✨ Key Features

- **Cinematic Guest Landing**: A visually stunning, immersive landing page with global travel flow visualizations.
- **AI-Driven Analytics**: Real-time insights into guest origin, market demand, and revenue metrics.
- **Smart Pricing Engine**: Automated rate adjustments based on seasonality, local festivals, and market trends.
- **AI Search & Discovery**: Interactive 3D globe for exploring destinations and sanctuaries.
- **Seamless Booking Flow**: A multi-step, glassmorphic booking experience with Razorpay integration.

## 🚀 Tech Stack

- **Frontend**: React 19 + Vite
- **Styling**: Vanilla CSS (Custom Raj Heritage Design System)
- **Database & Auth**: Supabase
- **Payments**: Razorpay
- **AI**: OpenRouter (GPT models)
- **Visuals**: amCharts 5 (Globe/Analytics), Lucide React

## 🛠️ Quick Start

For those familiar with Node.js development:

1. **Install**: `npm install`
2. **Configure**: Set up your `.env` file (see [CLIENT_GUIDE.md](./CLIENT_GUIDE.md) for details)
3. **Develop**: `npm run dev`
4. **Build**: `npm run build`
